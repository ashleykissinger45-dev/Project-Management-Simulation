# Project Management Simulation

A comprehensive project management simulation designed to replicate real-world PM workflows. This interactive application allows you to practice managing software development projects with realistic challenges including missed deadlines, team conflicts, and scope changes.

## Quick Start

```bash
python simulation_cli.py
```
